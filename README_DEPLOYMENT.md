# Obulapuram Sumanth — Professional Portfolio Website

This package contains a complete static portfolio website for a .NET Backend Developer profile.

## Included files

- `index.html` — main portfolio page
- `assets/css/style.css` — responsive premium styling
- `assets/js/main.js` — mobile menu, active navigation, reveal animations, contact mailto form
- `assets/img/profile.jpg` — profile image extracted from the existing portfolio, when available
- `assets/resume/Sumanth_Obulapuram_Resume.pdf` — downloadable resume
- `assets/favicon.svg` — custom favicon

## How to run locally

Open `index.html` directly in your browser.

## Free hosting options

### GitHub Pages

1. Create a new GitHub repository, for example `sumanth-portfolio`.
2. Upload all files from this folder.
3. Go to repository **Settings → Pages**.
4. Select branch `main` and folder `/root`.
5. Save and open the generated GitHub Pages URL.

### Netlify

1. Go to Netlify.
2. Drag and drop this complete folder into Netlify Deploys.
3. Netlify will publish the site and provide a live URL.
